# DICE ROLLER GAME

An interactive web-based dice roller where users choose how many dice to roll, and each die displays a random face. Built with **HTML**, **CSS**, and **JavaScript** for a fun and dynamic experience.

## Features
- User can select the number of dice to roll.
- Each die shows a random face from 1 to 6.
- Fun and easy-to-use interface.

## How to Use
1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Enter the number of dice you want to roll.
4. Click the **Roll Dice** button to see the results.

## Technologies Used
- HTML
- CSS
- JavaScript
